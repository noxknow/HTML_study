Reset 버튼

input type = "reset"은 모든 양식을 기본값으로 재설정하는 버튼을 정의합니다.




색상 선택기 표시

input type = "color"는 색상을 포함하는 입력 필드에 사용됩니다.

날짜 필드

input type = "date"는 날짜를 포함해야 하는 입력 필드에 사용됩니다.

현지 날짜 필드

input type = "datetime-local"는 시간대 없이 날짜 및 시간 입력 필드를 지정합니다.

제출 버튼으로 이미지 사용





파일 업로드

업로드할 파일을 선택할 수 있는 파일 선택 필드 표시



month 필드

input type = "month"를 사용하면 월과 연도를 선택할 수 있습니다.


전화 필드

input type = "tel"은 전화번호를 입력하는 입력 필드에 사용됩니다.



형식 : 010-1234-5678

시간 입력 컨트롤 표시

input type = "time"을 사용하면 사용자가 시간을 선택할 수 있습니다.

브라우저가 type = "time"을 지원하는 경우 입력 필드를 입력할 때 시간 선택기가 팝업 됩니다.


URL 입력 필드 표시

input type = "URL"은 URL 주소를 포함하는 입력 필드에 사용됩니다.

주간 입력 컨트롤 표시

input type = "week"을 통해 사용자는 주와 연도를 선택할 수 있습니다.

브라우저내에서 지원하는 경우 입력 필드를 입력할 때 날짜 선택기가 팝업 됩니다.
