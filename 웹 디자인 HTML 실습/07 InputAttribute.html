input readonly

raedonly 속성은 입력 피드가 읽기 전용이어야 함을 지정합니다.(변경할 수 없음)




input disabled

disabled 속성은 입력 필드가 비활성화되어야 함을 지정합니다.(사용할 수 없고 클릭 할 수 없음)




input size

size 속성은 입력 필드의 너비를 지정합니다.




input maxlength

maxlength 속성은 입력 필드에 허용되는 최대 문자 수를 지정합니다.




input min & max

min & max 속성은 입력 요소의 최솟값과 최대값을 지정합니다.





input placeholder

placeholder 속성은 사용자가 입력 필드의 예상 값을 설명하는 짧은 힌트를 지정합니다.



input step 속성

step 속성은 입력 요소의 유효한 숫자 간격을 지정 합니다.
